How to Run The App

## Server

# Development

1. go to folder server with terminal do this "cd ./server",
2. run "npm install"
3. run "nodemon bin/www.js"

# Testing

1. go to folder server with terminal do this "cd ./server",
2. run "npm install"
3. run "npm run test"

## Client

1. open new terminal in vscode terminal and go to folder client with "cd ./client"
2. run "npm install"
3. run "npm run dev"
